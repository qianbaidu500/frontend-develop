1. HTML 基础结构

HTML（HyperText Markup Language）是用 标签 描述网页结构的标记语言。一个最小可工作的 HTML 文档如下：

<!doctype html>
<html lang="zh-CN">
  <head>
    <meta charset="utf-8">
    <meta name="viewport" content="width=device-width, initial-scale=1">
    <title>页面标题</title>
  </head>
  <body>
    你好，世界！
  </body>
</html>
提示：<head> 放元数据（编码、标题、样式、脚本等），<body> 放页面可见内容。

2. 常用标签

文本与标题

<h1>~<h6> <p> <span> <strong> <em>

<h1>主标题</h1>
<p>一段文字，包含 <strong>强调</strong> 和 <em>倾斜</em>。</p>

链接与图片

<a> href target <img> alt

<a href="https://example.com" target="_blank">访问站点</a>
<img src="cat.png" alt="一只可爱的猫">

列表

<ul> <ol> <li>

<ol>
  <li>第一步</li>
  <li>第二步</li>
</ol>

分区与容器

<div> <section> <article>

<section>
  <h2>版块标题</h2>
  <p>内容...</p>
</section>

3. 语义化标签

语义化让结构更清晰、利于 SEO 与辅助技术。常见语义元素：

<header> <nav> <main> <section> <article> <aside> <footer> <figure> <figcaption> 

<article>
  <header>新闻标题</header>
  <p>正文...</p>
  <footer>作者与时间</footer>
</article>

4. 表单基础

表单用于收集用户输入。常用控件：

<input type="text|email|password|number|date"> <textarea> <select> <label for> required placeholder 

<form action="/submit" method="post">
  <label for="email">邮箱</label>
  <input id="email" name="email" type="email" required placeholder="you@example.com">

  <label for="role">角色</label>
  <select id="role" name="role">
    <option>学生</option>
    <option>工程师</option>
  </select>

  <button type="submit">提交</button>
</form>
可用性：为每个控件配对 <label>，为图片提供 alt，给表单元素添加 name。

5. 表格基础

<table>
  <caption>成绩表</caption>
  <thead>
    <tr><th>姓名</th><th>分数</th></tr>
  </thead>
  <tbody>
    <tr><td>小明</td><td>95</td></tr>
    <tr><td>小红</td><td>88</td></tr>
  </tbody>
</table>
使用 <thead> / <tbody> / <tfoot> 提高结构清晰度；用 scope 标明表头关联。

6. 布局与盒模型

盒模型

每个元素都是一个盒子：内容 content + 内边距 padding + 边框 border + 外边距 margin。

/* 让宽高更直观 */
*{ box-sizing: border-box; }

块级 vs 内联

  • 块级（如 div、section）独占一行，可设宽高。
  • 内联（如 span、a）不独占一行，不可直接设置宽高。

Flex 快速布局

.row { display:flex; gap:12px; }
.row .item { flex:1; }

Grid 网格布局

.grid { display:grid; grid-template-columns: repeat(3, 1fr); gap:12px; }

7. CSS 基础

引入方式

<!-- 行内 -->
<h1 style="color:tomato">标题</h1>

<!-- 内嵌 -->
<style> h1 { color: tomato; } </style>

<!-- 外链 -->
<link rel="stylesheet" href="styles.css">

选择器与优先级（从低到高）

  • 类型选择器（div、p）
  • 类选择器（.btn）
  • ID 选择器（#app）
  • 行内样式（style="..."）
  • !important（慎用）

常用属性

color background padding margin border display position flex/grid

8. JavaScript 基础

引入方式

<script>console.log('内联脚本')</script>
<script src="app.js" defer></script>

简单交互示例

<button id="hi">点我试试</button>
<script>
  document.addEventListener('DOMContentLoaded', () => {
    const btn = document.getElementById('hi');
    btn.addEventListener('click', () => {
      alert('你好，HTML！');
    });
  });
</script>

9. 可访问性与元信息

  • 为图片提供 alt 文本；为表单控件配对 <label for>。
  • 为页面设置 lang（本页已设置 zh-CN）。
  • 链接文本应描述目标，而非“点此”。
  • 合理使用语义标签，有助于屏幕阅读器理解结构。
  • 在 <head> 中添加 <meta name="description">、favicon 等。

10. 最佳实践清单

  • 始终声明 <!doctype html> 与正确编码 UTF-8。
  • 移动端适配：meta viewport。
  • 结构语义化，保持标题层级有序（仅一个 h1）。
  • 样式与脚本尽量外链，启用压缩与缓存。
  • 图片优化：合理尺寸、alt 文本、可使用 loading="lazy"。
  • 保证可访问性：对比度、键盘可达性、表单标签。
  • 调试工具：使用浏览器 DevTools 检查结构与样式。
