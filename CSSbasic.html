CSS 基础知识一页通

本页面覆盖选择器、盒模型、布局、文本样式、颜色、单位、层叠与优先级，并附带练习区。

1. CSS 的三种引入方式

/* 行内样式 */
<h1 style="color:red">标题</h1>

/* 内嵌样式 */
<style>
  h1 { color: red; }
</style>

/* 外部样式 */
<link rel="stylesheet" href="style.css">

2. 常用选择器

元素选择器 类 .class ID #id 子代 A > B 后代 A B 伪类 :hover 伪元素 ::after 

p { color: #333; }
.button { padding: 8px 12px; }
#header { background: #fafafa; }
ul li { margin-bottom: 8px; }
button:hover { background: #ddd; }
p::first-line { font-weight: bold; }

3. 盒模型 Box Model

元素由 content → padding → border → margin 组成。

.box {
  width: 200px;
  padding: 16px;
  border: 2px solid #333;
  margin: 20px;
}

推荐使用：

* { box-sizing: border-box; }

4. 常用属性分类

尺寸与边距

width: 100px;
height: 50px;
padding: 10px 20px;
margin: 20px auto;

边框与圆角

border: 1px solid #444;
border-radius: 8px;

背景

background-color: #f0f0f0;
background-image: url(bg.png);
background-size: cover;

文本样式

font-size: 16px;
font-weight: 600;
text-align: center;
line-height: 1.8;

5. 布局：Flex 与 Grid

Flex 适合一维布局

.row {
  display: flex;
  gap: 10px;
  justify-content: space-between;
  align-items: center;
}

Grid 适合二维布局

.grid {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  gap: 12px;
}

6. 层叠与优先级（从低到高）

  • 元素选择器（p, div）
  • 类选择器（.box）
  • ID 选择器（#main）
  • 行内样式（style=""）
  • !important（最高，不推荐随便使用）

7. 颜色与单位

颜色

color: #333;
color: rgb(255, 0, 0);
color: hsl(210, 50%, 50%);

单位

px（像素）
rem（相对根元素字体）
em（相对当前元素字体）
%（相对父元素）
vw/vh（相对视口尺寸）
